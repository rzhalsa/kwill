# Kwill
Kwill is a webapp which can generate, store, and update user-created smart D&D character sheets.

Once released, Kwill will support the following features:
- Generating user-specified smart D&D character sheets automatically. Smart character sheets will support features such as:

    - Fields being automatically populated with calculated values.

    - Tooltips on mouse hover for features and spells.

    - Built-in tools to aid in tasks such as skill checks, attacks, rests, or level ups.

- Allowing users to view all their created character sheets on their account dashboard.

- Allowing users to export a character sheet locally to their own machine.

- Allowing users to import a locally stored character sheet to the webapp.

---

### Access
Once released, Kwill will be able to be accessed by navigating to https://character-kwill.com on your internet browser of choice.
